# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter Web App** built using **HTML, CSS, and JavaScript**.  
It allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin**.

---

## 🚀 Features

- ✅ Input field for temperature  
- ✅ Dropdown to select unit (°C / °F / K)  
- ✅ Convert button  
- ✅ Displays converted results instantly  
- ✅ Input validation (handles invalid input)  
- ✅ Clean and responsive UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
